# css-inherit
css-inherit is a command-line version of [rework-inherit](https://github.com/reworkcss/rework-inherit/).

## Install
``` sh
$ npm install -g michaelrhodes/css-inherit
```

### Usage
#### cli
``` sh
$ cat some.css | css-inherit > build.css
```
